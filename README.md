# epubeeDownloader
自动下载epubee电子书。

## 使用方法
1. 安装NODE

。。。

2. 如果有git，使用git下载程序

`git clone https://github.com/GeorgeChen-666/epubeeDownloader.git`

3. 如果没有git，直接下载解压

。。。

4. 进入文件夹安装相关包

`npm i`

5. 记录你的epubeeid然后运行程序

`node index.js yourId`
